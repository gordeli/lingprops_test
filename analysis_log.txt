REPETITION ANALYSIS: How word frequency affects concreteness
======================================================================

where C(n,k) is the binomial coefficient, depth is WordNet hypernym depth,
and frequency is the word occurrence count.

LIMITING CASES
==============

Case 1: Single occurrence (frequency=1)
  log(C(depth + 1, 1)) = log(depth + 1)
  depth= 1: log(2) = 0.693147
  depth= 3: log(4) = 1.386294
  depth= 5: log(6) = 1.791759
  depth= 8: log(9) = 2.197225
  depth=12: log(13) = 2.564949

Case 2: Many repetitions (frequency >> depth)
  log(C(depth + f, f)) ~ depth * log(f) for large f

NORMALIZATION IMPACT
====================

Each repetition increases both the numerator (score) AND denominator (count)

KEY INSIGHT: Normalized score DECREASES with repetitions.
Repeating the same word yields diminishing returns per-word.
This is because log(C(d+f,f))/f is a monotonically decreasing function of f.
